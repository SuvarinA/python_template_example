# python_template_example
Example python for using template

1. Create a template file first called "settings.yml.template".
2. Write a python code and save it as "StringTemplate.py" to use the template file and input the configuration parameters in the python code.
3. Run the python script using "python StringTemplate.py". The console should provide the result of the program. And also provide the new file called "settings.yml".

Result:
dev:
  ip_addresses:
    - 10.0.0.101
    - 10.0.0.102
    - 10.0.0.103
  domain_name: dev.suvarinalibert.com
